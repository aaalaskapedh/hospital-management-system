1. File Naming
Use PascalCase for class-related files
Keep .h and .cpp names identical
Examples:Patient.h / Patient.cpp, Doctor.h / Doctor.cpp, Appointment.h / Appointment.cpp, Billing.h / Billing.cpp

2. Class Naming
Use PascalCase
Examples:
Patient,Doctor,Appointment,FileHandler,RecordManager

3. Function Naming
Use camelCase
Examples:bookAppointment(),displayAppointments(),addPatient(),deleteRecord()

4. Variable Naming
Use camelCase
Examples:patientID,doctorName,appointmentDate,recordCount

5. Constants Naming
Use UPPER_CASE
Examples:MAX_RECORDS,FILE_PATH,DEFAULT_ID

6. File Path Naming
Use lowercase folder names
Example:data/appointments.txt,data/patients.txt

7. Header Guards
Use uppercase with underscores
Example:#ifndef APPOINTMENT_H,#define APPOINTMENT_H
